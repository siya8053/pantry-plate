# 🍽 Pantry Plate – Recipe Generator

**Pantry Plate** helps you turn a single ingredient into a delicious meal.  
Perfect for students, office workers, or anyone with a busy schedule – just type what you have, and get recipe ideas instantly. Save your favorites so you never lose them.

> **Built with AI assistance** – I designed the idea, logic, and user experience, while using AI tools to help with implementation. This project reflects my vision for a simple, practical cooking companion.


## ✨ Features

- 🔍 **One‑ingredient search** – e.g., `egg`, `pasta`, `paneer`, `potato`
- 🥗 **Diet filters** – Vegetarian / Non‑vegetarian
- 🍔 **Style filters** – Healthy / Indulgent
- ❤️ **Favorites** – Save recipes to your browser’s local storage
- 📖 **Full recipe view** – Ingredients, step‑by‑step method with timings

🧠 How It Works
1.You enter an ingredient (e.g., “paneer” or “egg”).

2.The backend matches it to a set of predefined recipes.

3.Filters (diet / style) narrow the results.

4.Recipes are displayed as cards – click to see full details.

5.Click the heart icon to save favorites – they stay in your browser.

...If your ingredient isn’t in the built‑in list, Pantry Plate will generate a generic recipe (e.g., “Rice Curry”) using a template.
